# Crypto Tool

A command-line cryptography tool built in Python that demonstrates 
two fundamental security concepts: Caesar Cipher encryption and 
SHA-256 hashing.

## Features

- **Encrypt** a message using a Caesar Cipher with a custom shift key
- **Decrypt** a Caesar Cipher message using the same shift key
- **Hash** any message using SHA-256 (one-way, irreversible)
- **Integrity Check** — compare two messages by their hash to detect changes

python main.py
